The sprite package contains specialized implementations of {@link com.golden.gamedev.object.Sprite sprites}.

The four {@link com.golden.gamedev.object.Sprite sprite} implementations are:

  • {@link com.golden.gamedev.object.sprite.AdvanceSprite} - An advanced implementation of an {@link com.golden.gamedev.object.AnimatedSprite}.
    • {@link com.golden.gamedev.object.sprite.VolatileSprite} - An {@link com.golden.gamedev.object.sprite.AdvanceSprite} designed to animate once and then disappear.
  • {@link com.golden.gamedev.object.sprite.InvisibleSprite} - A {@link com.golden.gamedev.object.Sprite} that can be set to disappear.
  • {@link com.golden.gamedev.object.sprite.PatternSprite} - A {@link com.golden.gamedev.object.Sprite} that has its rendered image controlled by a delegate {@link com.golden.gamedev.object.Sprite pattern sprite}.
